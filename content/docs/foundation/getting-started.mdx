# Getting Started with NextChat

Welcome to NextChat! This guide will help you get up and running quickly.

## What is NextChat?

NextChat is a modern, AI-powered chat application built with Next.js. It provides:

- **AI Chat Interface** - Chat with various AI models
- **Plugin System** - Extend functionality with plugins
- **Project Management** - Built-in project management system
- **Documentation System** - Comprehensive documentation platform

## Quick Setup

### Prerequisites

- Node.js 18+ 
- Yarn package manager
- Git

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd nextchat
```

2. Install dependencies:
```bash
yarn install
```

3. Start the development server:
```bash
yarn dev
```

4. Open your browser to `http://localhost:3000`

## First Steps

1. **Configure AI Models** - Set up your preferred AI models in settings
2. **Explore Plugins** - Browse and install useful plugins
3. **Create Projects** - Start managing your development projects
4. **Read Documentation** - Explore the comprehensive documentation
